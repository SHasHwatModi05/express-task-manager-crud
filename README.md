# 🚀 Express Task Manager (CRUD App)

A full-featured **Task Manager Web Application** built using **Node.js, Express, EJS, and Tailwind CSS**.
This app allows users to create, read, update, and delete tasks with a clean UI and file-based storage.

---

## 🌐 Live Demo

👉 https://express-task-manager-crud.onrender.com/

---

## ✨ Features

* ✅ Create new tasks
* 📖 View all tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* 📂 File-based storage system
* ⚡ Fast and lightweight
* 🎨 Clean UI with Tailwind CSS

---

## 🛠 Tech Stack

* **Backend:** Node.js, Express.js
* **Frontend:** EJS, Tailwind CSS
* **Storage:** File System (fs module)
* **Tools:** Git, GitHub

---
